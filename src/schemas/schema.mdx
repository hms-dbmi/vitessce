# JSON Schema

JSON Schema is used to validate each response we receive from the (fake) API.
If a response fails validation, there will be a warning in the UI,
and a more informative error in the console.

(The summaries below are intended to be concise and human-readable;
Consult the [source code](https://github.com/hms-dbmi/vitessce/tree/master/src/schemas) if they aren't.)
